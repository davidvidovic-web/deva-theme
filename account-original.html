Deva Theme Button Components

1. Deva Themed Green Button - Color Change on Hover

This button changes from the standard Deva green (#48733d) to a brighter green (#15803d) on hover.

CSS Class: deva-btn deva-btn-hover-color

2. Deva Themed Green Button - Box Shadow Pop-out Effect

This button adds a prominent box shadow and lifts upward to create a "popping out" effect on hover.

CSS Class: deva-btn deva-btn-hover-shadow

3. Deva Themed Green Button - Scale Transform on Hover

This button scales up by 10% (scale(1.1)) on hover for a modern interaction effect.

CSS Class: deva-btn deva-btn-hover-scale

Usage Instructions

To use these buttons in your Deva theme, copy the CSS styles above and apply the appropriate classes:

HTML Structure:

<button class="deva-btn deva-btn-hover-color">Your Button Text</button>

<button class="deva-btn deva-btn-hover-shadow">Your Button Text</button>

<button class="deva-btn deva-btn-hover-scale">Your Button Text</button>
